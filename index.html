Política de Privacidade - Quinze

Última atualização: 23/03/2026

1. Coleta de Dados

Dados Financeiros: Todas as transações e planejamentos são armazenados localmente no seu dispositivo. O aplicativo Quinze não coleta nem envia esses dados para servidores externos.

Backup (Google Drive): O acesso ao Google Drive é utilizado apenas para permitir que o próprio usuário realize backups em sua conta pessoal.

2. Permissões Sensíveis

Leitura de Notificações: Utilizada para identificar automaticamente gastos em tempo real. Essas informações são processadas apenas no dispositivo e não são compartilhadas.

Biometria: Utilizada exclusivamente para autenticação local no aplicativo.

3. Serviços de Terceiros

O aplicativo pode utilizar serviços de terceiros que podem coletar informações:

  • AdMob (Google): Utilizado para exibir anúncios. Pode coletar identificadores de dispositivo e dados de uso para personalização de anúncios.
  • Firebase (Google): Utilizado para coleta de dados de falhas e desempenho (crash analytics), ajudando a melhorar a estabilidade do aplicativo.

4. Compartilhamento de Dados

O Quinze não compartilha dados pessoais dos usuários, exceto quando necessário para funcionamento dos serviços de terceiros mencionados.

5. Segurança

Adotamos medidas para proteger suas informações, porém nenhum sistema é 100% seguro.

6. Alterações nesta Política

Esta Política pode ser atualizada periodicamente. Recomendamos revisá-la regularmente.
